# Happy Birthday Joalan

An interactive, mobile-optimized birthday celebration website built specifically for Joalan. This project features a staged celebration experience with animations, music, and a personalized birthday letter.

## Live Demo
Experience the celebration here: [tasdidnoor.com/HappyBirthday/](https://tasdidnoor.com/HappyBirthday/)

## Features
- Staged Celebration: An interactive "Start" sequence that triggers music, confetti, and animations.
- Personalized Visuals: Custom-styled birthday banners, a festive cake, and a dynamic name display for Joalan.
- Mobile Optimized: Fully responsive design with media queries to ensure a perfect experience on laptops, tablets, and smartphones.
- No-Scroll Layout: A compact, single-screen design for the main party to keep all elements visible at once.
- Birthday Letter: A dedicated page featuring a heartfelt letter with a side-by-side text and image layout.
- Interactive Elements:
  - Snowflakes: Interactive falling snowflake effects.
  - Confetti: High-performance canvas-based confetti explosion.
  - Typewriter Effect: Animated birthday greetings.
  - Music: Integrated background track to set the mood.

## Technology Stack
- Frontend: HTML5, CSS3 (Bootstrap 4 for grid layout)
- Interactivity: JavaScript (jQuery, Typed.js, Snowflakes.js)
- Animations: CSS3 Keyframes and HTML5 Canvas

## Project Structure
- index.html: The main celebration hub.
- letter.html: The personalized birthday letter page.
- style.css: Custom styling and mobile optimizations.
- js/: Core logic for animations and effects.
- img/: Festive assets including the cake, banners, and balloons.

## How to Customize
1. Clone the repository: git clone https://github.com/tasdidnoor/HappyBirthday.git
2. Update the message in letter.html to add your personal touch.
3. Replace the image placeholder in letter.html with a photo of Joalan.
4. Host it anywhere (GitHub Pages, Vercel, or your personal domain).
