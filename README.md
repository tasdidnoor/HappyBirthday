# Minimalist Gold & White Birthday Celebration

A modern, high-end, and interactive birthday page designed with an "Apple-inspired" aesthetic. This project focuses on a premium, clean experience that feels like the future—minimalist, trustworthy, and high-quality.

## Features
- **Exquisite Design:** A clean white and soft gold palette featuring minimalist shadows and glassmorphism (blur) effects.
- **Interactive "Gold Dust":** A fullscreen entry screen with soft, floating gold particles that move subtly behind a glass-styled "Open Celebration" button.
- **Premium Typography:** Utilizes **Playfair Display** (a classic serif) for headers and **Inter** (modern sans-serif) for messages to ensure a sophisticated feel.
- **Sleek Animations:** 
  - Smooth transitions between the loader and main content.
  - Interactive **Gold, Silver, and White** confetti system.
  - Minimalist animated SVG cake with a glowing candle.
- **Dynamic Messaging:** 
  - Integrated `Typed.js` for an elegant, typewriter-style birthday wish.
  - Supports dynamic name delivery via URL parameters (e.g., `index.html?name=John`).
- **Interactive Letter:** A dedicated, clean space for a longer heartfelt birthday message.
- **Seamless Audio:** Integrated celebratory music that starts when the user "opens" the celebration.

## Technical Details
- **Frontend:** Vanilla HTML5 / CSS3 (No Frameworks).
- **Styling:** Custom Vanilla CSS for maximum flexibility and performance.
- **Interactivity:** jQuery for DOM manipulation and Typed.js for text animations.
- **Visuals:** Canvas-based particle systems for both the entry screen and the celebration confetti.

## How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. (Optional) Add `?name=RecipientName` to the URL to customize the name on the fly.

## Project Vision
To create a digital celebration that feels sophisticated, personal, and premium—moving away from traditional cluttered designs toward a refined, cinematic experience.
